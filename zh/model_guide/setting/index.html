# 软件和格式

我们使用的软件是Blockbench，您可以通过谷歌搜索下载 (opens new window)。 在建造模型时，我们推荐使用自由模型格式，您可以在后续将自由模型格式转换为基岩版格式。 自由模型格式的优点在于拥有基岩版所有功能的同时，支持多张贴图。 但请注意，未来我们在游戏中使用的模型格式依然是基岩版模型格式，所以在转换模型前请合并所有贴图。

# 插件

推荐的插件是Shape Generator和Cameras。使用该插件可以生成XYZ三轴的实心、空心的八边形和十六边形。

进入“文件/插件”，搜索Shape Generator并安装它。

Cameras插件将在决定该枪械具体游戏视角和物品展示方面起作用，详细操作将在后文解释。
